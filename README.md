# 💰 Smart Bill Split Calculator

A modern and responsive **Smart Bill Split Calculator** built using **HTML, CSS, and JavaScript**. This application helps users quickly calculate the total bill, tip amount, and individual share while offering multiple useful features like bill history, theme switching, receipt download, and multi-currency support.

---

## 🚀 Live Demo

https://your-username.github.io/Smart-Bill-Split-Calculator/


## ✨ Features

- 💸 Calculate total bill with custom tip percentage
- 👥 Split the bill among multiple people
- 💰 Multi-currency support
  - ₹ INR
  - $ USD
  - € EUR
  - £ GBP
- ⚡ Quick tip selection buttons (5%, 10%, 15%, 20%, 25%)
- 🌙 Dark Mode / Light Mode
- 📝 Stores last calculation using Local Storage
- 📜 Bill History (stores up to 10 recent calculations)
- 🗑️ Clear History option
- 📄 Download bill receipt as a text file
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism UI

## ⚙️ How It Works

1. Select your preferred currency.
2. Enter the total bill amount.
3. Enter a custom tip percentage or select one of the quick tip buttons.
4. Enter the number of people sharing the bill.
5. Click **Calculate**.
6. The application displays:
   - Bill Amount
   - Tip Amount
   - Total Amount
   - Amount each person pays
7. Download the receipt or review previous calculations from the history section.

---

## 💾 Local Storage Features

The application stores:

- Selected theme (Dark/Light)
- Last bill calculation
- Up to 10 recent bill calculations

This allows users to continue from where they left off even after refreshing the page.
